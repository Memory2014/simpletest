# SimpleRegionRestrictionCheck

## 支持OS

仅在Debian最新稳定版测试通过，其他OS未经测试，如运行出现问题请尝试解决依赖。

## 使用方法



For Ubuntu / Debian / Kali / Linux Mint
You'll need to update your package list first to ensure you're getting the latest version.
````bash
sudo apt update && sudo apt install curl
````

**使用脚本前请确认curl已安装**

````bash
bash <(curl -L -s https://github.com/Memory2014/simpletest/raw/main/simpletest.sh)
````
